Gulp boilerplate

A basic boilerplate using...

  • Gulp
  • Nunjucks
  • Sass
  • Scss-lint
  • Autoprefixr
  • Browserify
  • BrowserSync
  • JSHint
  • Imagemin
  • Dev/Prod env
  • Reporting
  • Deploy to gh-pages

Tasks

  1. gulp - Default task to build and run server
  2. gulp prod - Minify everything to prepare for deploy
  3. gulp deploy - Deploy to gh-pages
